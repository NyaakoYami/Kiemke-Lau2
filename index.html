📊 Xuất sang Google Sheets

File CSV đã được tải xuống. Làm theo các bước sau để import vào Google Sheets:

  1. Mở Google Sheets (sheets.google.com)
  2. Tạo bảng tính mới hoặc mở bảng hiện có
  3. Vào menu File → Import
  4. Chọn tab Upload → tải file CSV vừa tải
  5. Chọn "Replace spreadsheet" hoặc "Insert new sheet"
  6. Nhấn Import data là xong!
