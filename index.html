☁️ Cài đặt đồng bộ Google Sheets

Làm theo các bước sau để bật đồng bộ tự động giữa các máy:

  1. Mở Google Sheets → tạo bảng mới → đặt tên tuỳ ý
  2. Vào menu Extensions → Apps Script
  3. Xoá code cũ, dán toàn bộ nội dung file Code.gs vào
  4. Nhấn 💾 Save → nhấn Deploy → New deployment
  5. Chọn type: Web app → Execute as: Me → Who has access: Anyone
  6. Nhấn Deploy → copy URL dán vào ô bên dưới

📊 Xuất sang Google Sheets

File CSV đã được tải xuống. Làm theo các bước sau để import vào Google Sheets:

  1. Mở Google Sheets (sheets.google.com)
  2. Vào menu File → Import
  3. Chọn tab Upload → tải file CSV vừa tải
  4. Nhấn Import data là xong!
